Linear Regression

Used to find Linear Relationship between dependent and one or more independent variables
Linear Regression is a Supervised Learning Algo => solves Regression problems where we try to predict a Continuous Variable

Linear Regression can be further classified into two types:

  1. Simple Linear Regression
  2. Multiple Linear Regression

1. Simple Linear Regression (SLR)

It is the simplest model in machine learning. It models the linear relationship between the independent and dependent variables.

This application of Linear Regression is based on the TV and Sales data

  • In data we have one Independent or Input Variable which represents TV, denoted by X

  • In data we have one Dependent or Output Variable which represents Sales,denoted by y
    I am trying to build a Linear Relationship between these variables. This linear relationship can be modelled by mathematical equation of the form:-

                   Y = 0  + 1X    -------------   (1)

In this equation:

  • X is Independent Variables
  • Y is Dependent Variables
  • 1 is Coefficient for independent variable
  • 0 is Constant Term
  • 0 and 1 are called Parameters of Model

For simplicity, compare above equation with basic linear equation of form:-

               y = mx + c       -------------   (2)
  • m = 1 => Slope of line
  • c = 0 => Intercept of line

In this Simple Linear Regression Model, I am trying to fit a line which estimates Linear Relationship between X and Y
So, the question of fitting reduces to estimating the parameters of the model 0 and 1

Ordinary Least Square Method

One can draw a scatter plot between X and y which shows relationship between them

My task is to find a line which best fits this scatter plot This Line will help us to predict value of any Target variable for any given Feature variable This line is called **Regression line**

We can define an error function for any line. Then, the Regression Line is one which minimizes error function, such an error function is also called a Cost function

Performing Simple Linear Regression

Linear Regression Equation y = c + m_1x_1 + m_2x_2 + ... + m_nx_n

  • $y$ is the response
  • $c$ is the intercept
  • $m_1$ is the coefficient for the first feature
  • $m_n$ is the coefficient for the nth feature

In our case: $y = c + m_1 \times TV$

$m$ = model coefficients or model parameters

Model Performance

Regression Metrics

To Evaluate Model Performance

For Regression Problems there are two ways to Compute Model Performance

  1. RMSE (Root Mean Square Error)
    • RMSE is standard deviation of residuals, it gives us standard deviation of unexplained variance by model \
    • It results in lower values of RMSE as lower values of RMSE indicate better fit of data

  1. R-Squared Value
    • (R2) Correlation explains strength of relationship between an independent and dependent variable
    • R-square explains to what extent variance of one variable explains variance of second variable. It may also be known as the coefficient of determination
    • If R2 of a model is 0.50, then approximately half of observed variation can be explained by model's inputs
    • Its value can become negative if our model is wrong

Residual analysis

  • Linear regression model may not represent data appropriately, model may be a poor fit to data

    • So we should validate our model by defining and examining residual plots

    • Difference between observed value of dependent variable (y) and predicted value (yi) is called residual and is denoted by e or error

    • Scatter-plot of these residuals is called residual plot

If data points in a residual plot are randomly dispersed around horizontal axis and an approximate zero residual mean, a linear regression model may be appropriate for data. Otherwise a non-linear model may be more appropriate

If we take a look at both generated Residual errors plot(below), we can clearly see that train data plot pattern is non-random. Same is the case with the test data plot pattern. So, it suggests a better-fit for a non-linear model

Checking for Overfitting and Underfitting

Will see training set score and test set score

  • Can expect training set score to be 0.7996 which is averagely good. So, model learned relationships quite appropriately from training data
  • Model performs good on test data as test score will be 0.8149
    • It is a clear sign of good fit/ balanced fit
    • Hence, we can validated our finding that linear regression model provides good fit to data

Underfitting:

  • Model underfitts training data when model performs poorly on training data
  • This is because model is unable to capture relationship between input examples (often called X) and target values (often called Y)

Overfitting:

  • Model overfitts training data when you see that model performs well on training data but does not perform well on evaluation data
  • This is because model is memorizing data it has seen and is unable to generalize to unseen examples
